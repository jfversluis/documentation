## Android

:::tip
For Classic Xamarin, you will need to add framework reference to `Mono.Android.Export`.  You can do this by editing your head android project and adding the following

```xml
<ItemGroup>
    <Reference Include="Mono.Android.Export" />
</ItemGroup>
```
:::
